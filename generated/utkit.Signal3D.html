utkit.Signal3D¶

class utkit.Signal3D(data, items=None, major_axis=None, minor_axis=None, **kwargs)[source]¶

Represents data from raster scans. Each point is represented by its 2-D coordinates (X, Y), and contains a time series signal with time base t. By convention, the last axis is reserved for the time signals, and the first two axes compose the (X,Y) pairs.

Since this is a specialized Ultrasound library, we make the following convention:
  • axis 0: The scan direction along the x-axis.
  • axis 1: The scan direction along the y-axis.
  • axis 2: The time series signals.

Reshaping¶

shift_axes(shift[, axes, inplace]) Applies shifting of a given axis.
scale_axes(scale[, axes, inplace]) Scale axes by a given amount.
skew(angle, skew_axis, other_axis) Obtains the values for the axes after skewing by a certain angle, along a given plane.
flatten() Flatten an array and its corresponding indices.
operate([option, axis]) Operate on the signal along a given axis.

Transforms¶

bscan([option]) Convenience method that return B-scans from Ultrasound Testing raster scans.
cscan([theta]) Specialized method for computing the C-Scans from Ultrasound Testing raster scans.
dscan([option]) Convenience method that return D-scans from Ultrasound Testing raster scans.
extract([option, axis]) Extracts a Signal2D according to a given option by slicing along a specified axis.

Attributes

x Convenience property to return X-axis coordinates as ndarray.
ts The mean sampling interval for each of the axes.
y Convenience property to return Y-axis coordinates as ndarray.
axes Return index label(s) of the internal NDFrame
z Convenience property to return t-axis coordinates as ndarray.
