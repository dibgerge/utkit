utkit.Signal2D.fft¶

Signal2D.fft(ssb=False, axes=None, shape=None, **kwargs)[source]¶

Computes the Fourier transform in two dimensions, or along a specified axis.

Parameters:
  • ssb (bool, optional) – Determines if only the single sided Fourier transform will be returned.
  • axes (int, array_like, optional) – The axes along which to compute the FFT.
Returns:

A new signal representing the Fourier transform.
Return type:

Signal2D

Note

Keyword arguments can be given to the the underlying Fourier transform function scipy.fftpack.fft2().
