utkit.Signal2D.__call__¶

Signal2D.__call__(index=None, columns=None, **kwargs)[source]¶

Interpolate the axes. This function used scipy.interpolate.griddata().

Parameters:
  • index (array_like) – New index values to compute the interpolation.
  • columns (array_like) – New columns values to compute the interpolation.
Returns:

A new Signal2D object computed at the new given axes values.
Return type:

Signal2D

Notes

Other keyword arguments are passed directly to the interpolation function scipy.interpolate.griddata().
