utkit.Signal2D.extract¶

Signal2D.extract(axis=1, option='max')[source]¶

Extracts a 1-D Signal depending on the given option.

Parameters:
  • axis (int, optional) – Axis along which to extract the Signal. Options are 0/’y’/’index’ or 1/’x’/’columns’.
  • option ({'max'}, optional) – Currently only the option max is supported. This returns the signal at the maximum point in the Signal2D.
Returns:

A new Signal object representing the extracted signal.
Return type:

Signal
