utkit.Signal.limits¶

Signal.limits(threshold=-20)[source]¶

Computes the index limits where the signal first goes above a given threshold, and where it last goes below this threshold. Linear interpolation is used to find the exact point for this crossing.

Parameters:threshold (float, optional) – Units is dB. The value used to compute the where the signal first rises above and last falls below.
Returns:A two element tuple representing the start_index and end_index of the signal.
Return type:start_index, end_index (tuple (2,))
