utkit.Signal.spectrogram¶

Signal.spectrogram(width, overlap=0, **kwargs)[source]¶

Computes the spectrogram (short-time Fourier transform) of the signal. This method uses the function scipy.signal.spectrogram().

Parameters:
  • width (float) – Substitute for the argument nperseg in the function scipy.signal.spectrogram(). Here, width has same units as index.
  • overlap (float, optional) – Substitute for the argument noverlap in the function scipy.signal.spectrogram(). Here, overlap was same units as *index.
Returns:

A Signal2D class representing the
Return type:

Signal2D

Note

For other supported keyword arguments, see the documentation of scipy.signal.spectrogram(). However, the two arguments nperseg and noverlap should not be used.
