utkit.Signal.operate¶

Signal.operate(option='')[source]¶

This is a convenience function for common operations on signals. Returns the signal according to a given option.

Parameters:(string, char) (options) – The possible options are (combined options are allowed) +——————–+————————————–+ | option | Meaning | +====================+======================================+ | ‘’ (Default) | Return the raw signal | +——————–+————————————–+ | ‘n’ | normalized signal by max amplitude | +——————–+————————————–+ | ‘e’ | signal envelop | +——————–+————————————–+ | ‘d’ | decibel value | +——————–+————————————–+
Returns:The new signal with the specified operation.
Return type:Signal
