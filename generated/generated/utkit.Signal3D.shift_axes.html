utkit.Signal3D.shift_axes¶

Signal3D.shift_axes(shift, axes=None, inplace=False)[source]¶

Applies shifting of a given axis.

Parameters:
  • shift (float, array_like) – The value to shift the axis. If this is an array, it should be the same size as axes.
  • axes (int, string, array_like, optional) – Specifies which of the axes to shift. To shift a single axis, axes is an int or string representing the dimension or the axis alias name. To shift multiple axes, axes can be an array of value or string names. If axes is None, all three axes are shifted.
  • inplace (bool) – Modify the current instance if True.
Returns:

A copy of Signal3D with shifted axes, or the current instance if inplace is True.
Return type:

Signal3D
