utkit.Signal3D.extract¶

Signal3D.extract(option='max', axis=0)[source]¶

Extracts a Signal2D according to a given option by slicing along a specified axis.

Parameters:
  • option ({'max', 'var', float} optional) – Select the method to find the slice * ‘max’: The maximum amplitude along the axis * ‘var’: the maximum variance along the axis. * scalar: A scalar can be specified to select a specific point along the axis
  • axis (scalar/string, optional) – The axis along which to extract the slice.
Returns:

A Signal2D object representing the slice.
Return type:

Signal2D
