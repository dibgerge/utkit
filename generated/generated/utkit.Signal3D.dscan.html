utkit.Signal3D.dscan¶

Signal3D.dscan(option='max')[source]¶

Convenience method that return D-scans from Ultrasound Testing raster scans. Slices the raster scan along the y-t axes (i.e. at a given x location).

Parameters:option ({'max', 'var', float}, optional) – Select the method to find the slice * ‘max’: The maximum amplitude along the axis * ‘var’: the maximum variance along the axis. * scalar: A scalar can be specified to select a specific point along the axis
Returns:Extracted D-scan as a Signal2D object.
Return type:Signal2D
