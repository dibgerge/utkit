utkit.Signal¶

class utkit.Signal(data=None, index=None, *args, **kwargs)[source]¶

Represents physical signals, by keeping track of the index (time/space) and corresponding values. This class extends the Pandas pandas.Series class to make it more convenient to handle signals generally encountered in ultrasonics or RF.

The class constructor is the same as that used for pandas.Series. In addition, if data is an array, and index is a scalar, then index is interpreted as the sampling time, and the signal basis will be constructed as uniformly sampled at intervals specified by the scalar index and starting from 0.

For example, to define a sine wave signal at 100 kHz and sampling interval of 1 microsecond,

import numpy as np
import utkit

Ts = 1e-6
t = np.arange(100)*Ts
s = utkit.Signal(np.sin(2*np.pi*100e3*t), index=t)

the last line which calls the constructor of Signal is also equivalent to:

s = utkit.Signal(np.sin(2*np.pi*100e3*t), index=Ts)

The class Signal provides various methods for signal reshaping, transforms, and feature extraction.

Reshaping¶

__call__([key, ts]) Make the Signal act like a function, where given index values are computed using an interpolant, and do not necessarily require to be at one of the current sample points.
remove_mean() Subtracts the mean of the signal.
window([index1, index2, is_positional, ...]) Applies a window to the signal within a given time range.
filter_freq(cutoff[, option, win_fcn]) Applies a frequency domain filter to the signal.
normalize([option]) Normalizes the signal according to a given option.
segment(thres, pulse_width[, win_fcn]) Segments the signal into a collection of signals, with each item in the collection, representing the signal within a given time window.
operate([option]) This is a convenience function for common operations on signals.

Transforms¶

coherence(other, width[, overlap]) Compute the short-time correlation coefficient of the signal.
fft([nfft, ssb]) Computes the Fast Fourier transform of the signal using scipy.fftpack.fft() function.
psd(width[, overlap]) Computes the periodogram of the signal.
spectrogram(width[, overlap]) Computes the spectrogram (short-time Fourier transform) of the signal.

Features¶

bandwidth([threshold, nfft]) Computes the bandwidth of the signal by finding the range of frequencies where the signal is above a given threshold.
center_frequency([threshold, nfft]) Computes the center frequency of the signal, which is the mean of the bandwidth limits.
energy([option]) Computes the energy of the given waveform in the specified domain.
limits([threshold]) Computes the index limits where the signal first goes above a given threshold, and where it last goes below this threshold.
maxof([option]) Computes the maximum of the signal according to a given method.
peak_frequency([threshold, nfft]) Computes the peak frequency of the signal.
tof([method]) Computes the time of flight relative to another signal.

See pandas.Series for more information about using the base class library, including tutorials and additional methods that are very useful to use in conjunction with this library.

Attributes¶

extent Get the signal index extent.
fs Get the signal sampling frequency.
ts Get the signal sampling period.
